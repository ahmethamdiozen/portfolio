## Problem

Creating professional presentations is time-consuming. Users spend hours structuring content, writing copy, and formatting slides. There was a gap for an editor that combined AI generation with human refinement in a seamless interface.

## Solution

Built a browser-based presentation editor with:
- AI outline generation from a single topic prompt
- Slide-by-slide content expansion with GPT-4
- Rich text editing via Tiptap with formatting controls
- Real-time streaming of AI-generated content
- Export to PDF and PPTX formats

The Vercel AI SDK handles streaming responses, making content generation feel instant. Users can accept, reject, or edit any AI suggestion inline.

## Result

- Average presentation creation time reduced from 2 hours to 20 minutes
- Streaming UI made generation feel responsive and interactive
- Export feature enabled use in existing presentation workflows
- Positive feedback on the hybrid AI + manual editing UX
